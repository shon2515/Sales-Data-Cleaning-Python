# Sales-Data-Cleaning (PYTHON)
End-to-end data cleaning project handling missing values, invalid entries, and data type conversion using Python(Pandas)



## 📌 Objective
Clean and preprocess raw sales transaction data for analysis.

## 🧹 Data Cleaning Steps
- Removed missing values (NaN)
- Handled invalid entries (UNKNOWN, ERROR)
- Converted data types (Quantity, Price, Total Spent, Transaction Date)
- Removed duplicates
- Sorted data by transaction date

## 🛠 Tools Used
- Python
- Pandas
- Google Colab

## 📂 Output
Cleaned dataset exported as Excel file.

## 💡 Key Skills
- Data Cleaning
- Data Preprocessing
- Handling Missing Values
- Data Type Conversion
